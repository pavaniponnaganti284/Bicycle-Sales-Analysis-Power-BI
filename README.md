# Bicycle-Sales-Analysis-Power-BI


📌 Description

This project extends an existing Power BI report by adding a new page called "Pie Charts" to analyze bicycle sales data using pie chart visualizations.

🚀 Features
Added a new report page: Pie Charts
Visualized sales distribution using pie charts
Applied interactive filters for better analysis


🛠️ Tools Used
Power BI Desktop

📄 Visualizations

🥧 Chart 1: Units Sold by State
Legend: State
Values: Sum of Units Sold


🥧 Chart 2: Units Sold by Product
Legend: Product
Values: Sum of Units Sold


🥧 Chart 3: Average Profit by State
Legend: State
Values: Average of Profit

⚙️ Steps
Open the existing Power BI report
Add a new page
Rename it to "Pie Charts"
Insert three pie charts
Assign fields to Legend and Values
Apply page-level filters
Save the report


📸 Output
Page 2: Pie Charts with 3 visualizations
Filters applied for Month Name and State
